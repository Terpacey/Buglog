How to use BugLog

  1. Select or create a Project and Build above.
  2. Go to Test Cases to log and run your test cases.
  3. Go to Defects to report bugs found during testing.
  4. Go to Metrics to view execution rates and defect trends.
  5. Go to Settings to customise status, priority, and severity values.
  6. Return here any time for a quick results summary.
